# Dictionary

The dictionary functionality enables users to find comprehensive details for the words.

## Features

### Search

1. User is able to search with fuzzy match regardless of special czech characters - e.g. search term: "pri" , result: "p≈ôidat"
2. The search results are shown with their type of word, gender(if any), person(if any), tense (if any), plurality (if any), person(if any)

### On click of search results

When a selection happens the autocomplete is hidden and instead the czech translation of the selected word is shown as a title. Then relevant information is shown as sections of cards.

1. 1st card shows english translation and type of word.
2. 2nd card shows the word forms and relevant informationm (form_type info, gender, plurality, etc.)
3. 3rd card shows examples of sentences in which the word is used.
